About the Project

This website was created for the JavaScript/jQuery class final. The portfolio features exercises and assignments completed during the Summer 2015 class Experimental Interaction.

Each of the assignment and exercises featured are created utilizing P5.js. The website showcases the various digital interaction topics covered, such as Computational Aesthetics, Visual Time, Emulating Nature, etc.
